# Linux Networking
### Commands
`ifconfig` show info about network interface
- shows devices connected to your machine
- `eth0` for ethernet
- `wlan0` for wireless
`ip -4 addr` shows your ipv4 address
`ping` checks response from a domain
`netstat` shows detail about network connectivity
`netstat -a` shows listenning ports
`curl` used to transfer data to or from a website - can be used to get the source code of a website to your terminal

```bash
route # show the ip routing table
hostname # show/set systems host name
```
`nmap` is a network exploration tool

used to scan networks
